# Nexara 文件系统架构技术分析报告

## 1. 概述

Nexara 采用了一个完整的、安全的、现代化的文件系统架构，基于 React Native 和 Expo 生态构建，具有以下核心特点：

- **安全性优先**：严格的沙箱机制和权限控制
- **物理同步**：内存状态与物理文件系统的双向同步
- **智能存储**：多种存储介质的合理利用
- **向量化支持**：内置 RAG 系统的文件处理能力

## 5. 权限管理和访问控制

### 5.1 沙箱安全机制

```typescript
const resolveSafePath = async (relativePath: string): Promise<string> => {
  // 1. 确保沙箱根目录存在
  const sandboxInfo = await FileSystem.getInfoAsync(SANDBOX_ROOT);
  if (!sandboxInfo.exists) {
    await FileSystem.makeDirectoryAsync(SANDBOX_ROOT, { intermediates: true });
  }

  // 2. 规范化路径：移除开头的 ./ 或 /
  let cleanPath = relativePath.replace(/^(\.\/|\/)+/, '');

  // 3. 检查非法字符 (如 ..)
  if (cleanPath.includes('..')) {
    throw new Error('Security Violation: Access to parent directory (..) is denied.');
  }

  // 4. 构建完整路径
  return SANDBOX_ROOT + cleanPath;
};
```

**安全措施：**
- 路径规范化处理
- 父目录访问禁止
- 沙箱边界严格控制
- 自动目录创建

### 5.2 操作权限分类

class DocumentProcessor {
  async processFile(uri: string, fileName: string, mimeType?: string) {
    const fileType = this.detectFileType(fileName, mimeType);
    
    switch (fileType) {
      case 'image': return this.processImage(uri, fileName);
      case 'pdf': return { content: await this.processPdf(uri), type: 'text' };
      case 'docx': return { content: await this.processDocx(uri), type: 'text' };
      case 'xlsx': return { content: await this.processXlsx(uri), type: 'text' };
      default: return { content: await this.processText(uri), type: 'text' };
    }
  }
}
