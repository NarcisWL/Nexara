## 测试环境准备

### 前置条件
- [ ] Android 设备或模拟器已连接
- [ ] 应用已成功编译并安装 (`npm run android`)
- [ ] 至少配置 1 个 API 服务商（建议 SiliconFlow 或 OpenAI）
- [ ] 至少配置 1 个 Embedding 模型
- [ ] 至少配置 1 个 Chat 模型（用于查询重写）
- [ ] 至少上传 3-5 个文档到知识库（建议不同主题）

### 测试数据准备
建议准备以下测试文档：
1. **技术文档**（如 React 教程）- 用于测试语义检索
2. **产品说明**（如手机参数）- 用于测试关键词检索
3. **FAQ 文档** - 用于测试混合检索
4. **长文档**（>5000字）- 用于测试分块和召回
