# 项目设置面板架构

## 四大设置面板对应关系

### 1. 全局设置面板
**用户称呼**: 全局设置面板  
**入口**: APP首页 → 底部导航栏"设置"按钮  
**组件文件**: 待确定（可能在 `src/app/(tabs)/settings.tsx` 或类似）  
**结构**: 
- Tab 1: 应用设置页（包含备份、通用配置等）
- Tab 2: 服务商管理页（API配置、模型设置等）

**RAG集成位置**: 应用设置页 Tab，新增"RAG配置"分组

**相关组件**:
- `BackupSettings.tsx` - 备份设置
- `ProviderSettings.tsx` - 服务商设置

---

### 2. 超级助手面板
**用户称呼**: 超级助手面板  
**入口**: 首页超级助手悬浮按钮 → 超级助手会话界面 → 右上角设置按钮  
**组件文件**: 待确定（可能与 SuperAssistantFAB 关联）  
**用途**: 超级助手专属设置

**RAG集成位置**: 新增"RAG配置"区域，使用 AgentRagConfigPanel 组件

**相关组件**:
- `SuperAssistantFAB.tsx` - 超级助手悬浮按钮
- `spa-store.ts` - 超级助手状态管理

---

### 3. 助手设置面板
**用户称呼**: 助手设置面板  
**入口**: 会话列表 → 右上角设置按钮  
**组件文件**: `ModelSettingsModal.tsx` (推测)  
**用途**: 管理所有助手的通用设置

**RAG集成位置**: 新增"RAG配置"标签页或区域

**相关组件**:
- `ModelSettingsModal.tsx` - 模型设置模态框
- `ModelPicker.tsx` - 模型选择器

---

### 4. 会话设置面板
**用户称呼**: 会话设置面板  
**入口**: 会话界面 → 右上角设置按钮  
**组件文件**: 待确定  
**用途**: 单个会话的设置（推理参数、RAG选项等）

**RAG集成**: 本次不涉及（已有 RagSettingsPanel）

**相关组件**:
- `InferenceSettings.tsx` - 推理设置
- `RagSettingsPanel.tsx` - RAG设置面板（会话级）

## RAG配置集成策略

### GlobalRagConfigPanel 使用位置
- 全局设置面板 → 应用设置 Tab → "RAG配置"分组

### AgentRagConfigPanel 使用位置
- 超级助手面板 → "RAG配置"区域
- 助手设置面板 → "RAG配置"标签页/区域

### RagDebugPanel 使用位置
- 全局设置面板 → 应用设置 Tab → "RAG配置" → "查看向量库统计"按钮
