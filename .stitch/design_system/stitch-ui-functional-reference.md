## 二、Group A — 应用外壳与全局导航

### A1 欢迎页 (`welcome`)

| 维度 | 描述 |
|------|------|
| **目的** | 首次启动引导页，用户选择语言后进入应用 |
| **UI 元素** | ① 居中 SVG Logo "Nexara"（笔画描边动画入场）② 副标题 "Select Language / 选择语言" ③ 两个语言按钮: "中文 (简体)" 和 "English"，glass-panel 风格 |
| **交互** | 点击语言按钮 → 保存语言偏好 → 跳转到聊天首页 |
| **数据** | 无（静态页面） |
| **状态** | 仅显示一次（首次安装后），后续启动跳过 |
| **设计要点** | 深色背景 `#131315`；Logo 使用 Manrope 粗体；语言按钮 `border-radius: 18px`，毛玻璃边框 `0.5px` |

### A2 底部 Tab 导航 (`tab-bar`)

| 维度 | 描述 |
|------|------|
| **目的** | 应用主导航，3 个 Tab 切换 |
| **UI 元素** | ① 底部固定毛玻璃栏 ② 3 个 Tab: Chat (MessageSquare 图标) / Library (Library 图标) / Settings (Settings 图标) ③ 每个 Tab: 图标 + 文字标签 |
| **交互** | 点击切换 Tab（淡入淡出过渡）；键盘弹出时自动隐藏 |
| **数据** | Tab 标签文本来自 i18n（中/英双语） |
| **设计要点** | 毛玻璃 `blur(20px) + rgba(0,0,0,0.6)`；高度 65px + 底部安全距离；激活态使用 `--accent` 色；顶部分隔线 `0.5px glass-border` |

### A3 全局加载屏幕 (`root-loading`)

| 维度 | 描述 |
|------|------|
| **目的** | 数据库初始化时的短暂加载画面 |
| **UI 元素** | ① 纯色背景 `#131315` ② 居中 ActivityIndicator（品牌色） |
| **交互** | 无 |
| **设计要点** | 极简，仅品牌色 Spinner |

## 三、Group B — 聊天 Tab 会话列表与 Agent 管理

### B1 助手列表首页 (`chat-home`)

| 维度 | 描述 |
|------|------|
| **目的** | 用户创建的所有 AI 助手列表，支持搜索、置顶、删除、新建 |
| **UI 元素** | ① 大标题 "对话" + 副标题 "你的智能助手团队" ② 右上角 "+" 按钮 (48×48px, glass-panel, border-radius: 16px) ③ 搜索栏（固定不随列表滚动，毛玻璃背景, border-radius: 12px, Search 图标 + placeholder） ④ Agent 列表项: 左侧头像 (40×40, border-radius: 12px, 品牌色背景) + 名称 + 描述 ⑤ 列表项间 0.5px glass-border 分隔线 ⑥ 右下角 FAB (56×56px, 圆形, 品牌色背景 + Sparkles 图标, 悬浮阴影) |
| **交互** | 点击 Agent → 进入该 Agent 的会话列表 (B2)；左滑 Agent 项 → 显示"置顶/删除"操作；点击 "+" → 创建新 Agent 并进入编辑器 (B3)；点击 FAB → 打开超级助手会话 (C1)；搜索实时过滤 |
| **数据** | Agent 列表 (名称、描述、头像色、头像图标、是否置顶)；按置顶优先 + 创建时间排序 |
| **状态** | 空状态: 无 Agent 时显示引导文案 + "创建第一个助手" 按钮 |
| **设计要点** | 搜索栏固定在顶部（不随列表滚动）；列表项 hover/press 使用 `scale(0.97) spring` 动画；左滑操作按钮带弹性回弹 |

### B2 Agent 会话列表 (`agent-sessions`)

| 维度 | 描述 |
|------|------|
| **目的** | 展示某个 Agent 下的所有历史会话 |
| **UI 元素** | ① GlassHeader 毛玻璃顶栏: 返回按钮 (40×40 圆形, glass 背景) + Agent 名称 + 会话数量 ② 右上角设置图标按钮 ③ 搜索栏 ④ 会话卡片列表: 会话标题 + 最后消息预览 + 时间 ⑤ 右下角 "+" FAB (Agent 品牌色) |
| **交互** | 点击会话 → 进入聊天 (C1)；左滑会话 → 置顶/删除；点击 FAB → 创建新会话并跳转；点击设置图标 → 跳转 Agent 编辑器 (B3) |
| **数据** | Agent 名称/头像/品牌色；会话列表 (标题、最后消息、时间、是否置顶) |
| **状态** | 空状态: 图标 + "还没有对话，开始第一个吧" |

### B3 Agent 编辑器 (`agent-edit`)

| 维度 | 描述 |
|------|------|
| **目的** | 编辑 AI 助手的全部配置：名称、描述、头像、颜色、系统提示词、模型、温度、RAG |
| **UI 元素** | ① GlassHeader "编辑助手" ② 基本信息: 名称输入框 + 描述多行输入框 ③ 外观区 (可折叠): 80×80 头像预览 + 10 个预设图标网格 + 自定义图片上传按钮 + ColorPickerPanel 颜色选择器 ④ 性格: 系统提示词预览卡片 (虚线边框, "已配置"/"未设置" 状态徽章) ⑤ 模型配置: 引擎选择器 (打开 ModelPicker) + InferencePresets 三卡片 (精确/均衡/创意) ⑥ RAG 配置入口 (跳转链接) ⑦ 高级检索入口 (跳转链接) ⑧ 危险区: 删除助手按钮 (红色) |
| **交互** | 编辑名称/描述 → 自动保存 (防抖)；选择预设图标/上传头像/切换颜色 → 实时更新头像预览；点击提示词区域 → 打开全屏文本编辑器；选择模型 → 打开 ModelPicker 底部弹窗；点击 InferencePresets 卡片 → 更新温度/TopP；跳转 RAG/高级检索子页面；删除助手 → 确认对话框 |
| **数据** | Agent 名称、描述、系统提示词、默认模型、温度、头像图标/图片、品牌色 |
| **设计要点** | InferencePresets 三卡片: 精确(紫色 Code 图标)、均衡(青色 Zap)、创意(琥珀色 BookOpen)；选中卡片有品牌色边框 + 着色背景；ColorPickerPanel 是一行预设色圆点 + 自定义 (RainbowSlider) |

### E2 提供商表单 (`provider-form`)

| 维度 | 描述 |
|------|------|
| **目的** | 新增/编辑 API 提供商配置 |
| **UI 元素** | ① GlassHeader "添加/编辑提供商" ② 预设网格 (2 列): 18+ 提供商卡片 (OpenAI/Anthropic/Gemini/DeepSeek 等品牌图标) ③ 表单字段: 名称 + Base URL + API Key (安全输入) + Google Cloud Project ID + Region + Service Account JSON ④ 底部保存按钮 (动画缩放) |
